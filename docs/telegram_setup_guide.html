AMEVA-WoL 초보자 완벽 가이드

텔레그램 앱 설치부터 봇 생성, User ID 확인, 실행 방법까지 쉽게 배우기

1. 텔레그램 설치 2. 봇 생성 (BotFather) 3. 내 User ID 확인 4. AMEVA-WoL 설정 & 실행 5. 텔레그램 명령 사용법
1
텔레그램(Telegram) 앱 설치하기

텔레그램은 컴퓨터 및 스마트폰에서 안전하게 제어 명령을 보낼 수 있는 무료 메신저입니다.

  • 스마트폰 (Android / iPhone): Google Play 스토어 또는 Apple App Store에서 Telegram을 검색하여 설치합니다.
  • PC (Windows / Mac / Linux): 공식 홈페이지(desktop.telegram.org)에서 PC용 텔레그램을 다운로드하여 설치합니다.

앱을 켠 후 본인의 전화번호로 회원가입 및 인식을 완료하세요.

2
@BotFather로 나만의 텔레그램 봇 만들기

텔레그램의 공식 봇 관리 계정인 @BotFather를 이용하여 나만의 봇을 발급받습니다.

  1. 텔레그램 상단 검색창에 @BotFather를 검색하고 파란색 인증 마크가 있는 계정을 선택합니다.
  2. 채팅방에 /newbot 명령어를 입력하고 전송합니다.
  3. 봇의 이름(Name)을 입력하라는 안내가 나오면 원하는 이름을 입력합니다. (예: My Home WoL)
  4. 봇의 아이디(Username)를 입력하라는 안내가 나오면 반드시 끝이 bot으로 끝나는 아이디를 입력합니다. (예: my_home_wol_bot)
  5. 생성이 완료되면 API Token (토큰)이 발급됩니다.
⚠️ 중요 보안주의!
발급받은 봇 토큰(API Token)은 비밀번호와 같습니다! 절대로 타인에게 공유하거나 인터넷에 올리지 마세요.

발급된 토큰 예시: 123456789:ABCdefGhIJKlmNoPQRsTUVwxyZ_ExampleToken

3
내 텔레그램 User ID (숫자 ID) 확인하기

아무나 내 봇 조종을 못하게 막기 위해 본인의 숫자 User ID를 확인하여 허용 목록에 등록해야 합니다.

  1. 텔레그램 검색창에 @userinfobot 또는 @raw_data_bot을 검색합니다.
  2. 해당 봇과의 대화창에 /start를 전송합니다.
  3. 봇이 응답으로 알려주는 Id: 123456789 형태의 숫자를 복사합니다.
💡 팁: 봇 생성 후 AMEVA-WoL을 실행한 뒤 본인 봇 채팅방에서 /id 명령어를 전송해도 확인 가능합니다.
4
환경 자가 진단 스크립트 실행 (A-Z 자동 점검)

AMEVA-WoL은 실행에 필요한 모든 설치 요소(파이썬, 라이브러리, .env 토큰, User ID, 네트워크, 데이터 폴더)를 A부터 Z까지 자동으로 전수 점검하고 부족한 항목에 대한 해결책을 조언해주는 진단 스크립트를 제공합니다.

• Termux / Linux (쉘 스크립트):

bash scripts/check-environment.sh

• Windows (파워쉘 스크립트):

powershell -ExecutionPolicy Bypass -File .\scripts\check-environment.ps1
💡 진단 결과 리포트 예시:
만약 .env 파일이나 패키지가 빠져있다면 [FAIL] .env file missing -> FIX: Copy-Item .env.example .env와 같이 즉시 해결 가능한 방법(Heuristic Fix)을 친절하게 알려줍니다.
5
텔레그램 봇으로 컴퓨터 제어하기

내가 만든 텔레그램 봇 채팅방에 들어가서 아래 명령어들을 입력합니다.

1) 내 컴퓨터 등록하기 (/add)

/add desktop AA:BB:CC:DD:EE:01 192.168.0.100
  • desktop: 내가 부를 이름 (알리프)
  • AA:BB:CC:DD:EE:01: 랜카드 MAC 주소
  • 192.168.0.100: 컴퓨터 내부 IP 주소

2) 컴퓨터 켜기 (/wake)

/wake desktop

등록된 모든 컴퓨터를 켜려면: /wake all

3) 컴퓨터 켜짐/꺼짐 상태 확인 (/status)

/status desktop

4) 등록된 목록 보기 (/list)

/list
